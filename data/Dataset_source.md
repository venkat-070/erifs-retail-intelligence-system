# Dataset Source

The dataset used in this project is the **Online Retail II Dataset**, a public retail transaction dataset commonly used for customer behavior analysis and retail analytics.

Due to GitHub file size limitations (43.5 MB), the dataset is not stored directly in this repository.

## 📁 Data Source
The raw dataset used for this project (`online_retail_II.xlsx`) is 43.5 MB, You can download the exact dataset I used directly from Kaggle:
[Download the Online Retail II Dataset Here](https://www.kaggle.com/datasets/mathchi/online-retail-ii-data-set)

After downloading the dataset, place the file inside the **data** folder to reproduce the analysis.

File name used in this project:

online_retail_dataset.xlsx
